# Feature Development

## Steps

1. Identify feature from backlog or discussion
2. Create a branch from main
3. Implement with tests
4. Open pull request
5. Address code review feedback
6. Merge to main

## Definition of Done

- Feature works as described
- Tests pass
- Code reviewed and merged
- Backlog item updated or closed
