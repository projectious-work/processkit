⚠ What happened with Claude's limits? Three factors converged around 1 April 2026:
  • Promotion ended: Anthropic ran a "Spring Break" 2× usage promotion (13–28 March) for off-peak hours. Once it expired, users felt limits returned to "normal" — which felt like a downgrade.
  • Peak-hour throttling: Anthropic quietly tightened 5-hour session limits during US business hours (5–11 AM PT) to manage a 30%+ month-over-month traffic surge after the OpenAI Pentagon-boycott migration.
  • Claude Code caching bugs: Users discovered bugs breaking prompt cache, silently inflating token consumption by 10–20×. Anthropic acknowledged limits were being "hit way faster than expected" and called it their top priority.

🗣️ Community Consensus — Who Uses What for Coding (April 2026)

Claude — "The Coding Partner"

  • Powers Cursor, Windsurf, and Claude Code — the dominant AI coding editors
  • Best at understanding ambiguous intent and producing clean, idiomatic code
  • Opus 4.6 described as "ruined all other models for me" on r/ClaudeAI
  • Pain point: opaque limits and post-promotion throttling frustrating power users

ChatGPT / Codex — "The All-Rounder"

  • Largest ecosystem: plugins, GPTs, Copilot integration
  • GPT-5.4 best for structured output, agent loops, and fast iteration
  • New $100 Pro 5× tier (9 Apr) directly targets Claude Max 5× — with 10× Codex promo thru May 31
  • Codex agent for autonomous coding. 3M+ weekly users. Native computer use.
  • Pain point: Plus "rebalanced" alongside upsell launch; trust in stated limits eroding

Gemini — "The Value Giant"

  • 1M context lets you dump entire codebases. Cheapest frontier API.
  • Jules coding agent improving rapidly. Google Workspace integration.
  • 3.1 Pro surprised everyone with 80.6% SWE-bench at budget pricing
  • Pain point: needs precise prompts; sometimes over-explains or under-explains

Mistral — "The EU Privacy Pick"

  • Cheapest premium sub at €14.99. No-Telemetry mode unique in market.
  • Codestral strong for code in Le Chat. Flash Answers blazing fast.
  • EU data residency, GDPR by design — ideal for European enterprises
  • Pain point: smaller ecosystem; not yet frontier on coding benchmarks

Grok — "The Wild Card"

  • 2M token context is largest in market. Real-time X/web data unique.
  • Grok 4 competitive on reasoning. Heavy mode uses multi-agent approach.
  • Confusing plan structure (X Premium vs SuperGrok vs standalone)
  • Pain point: safety controversies; youngest ecosystem; no dedicated code agent

DeepSeek / Open-Weight — "The Disruptor"

  • V4 at $0.28/$1.10 per M tokens — 27× cheaper than closed frontier
  • MiniMax M2.5 hits 80.2% SWE-bench as open-weight — near Claude Opus
  • Self-hostable for full data sovereignty. Ideal for your Vast.ai setup.
  • Pain point: China jurisdiction concerns; smaller community support

Gemma 4 — "The Local-First Powerhouse" NEW Apr 2

  • 31B Dense: 80% LiveCodeBench, 89% AIME, 86% τ2-bench agentic. Fits BF16 on 2×A6000 (~62GB). No quant needed.
  • 26B MoE: Only 3.8B active params → 80–150+ tok/s on consumer GPUs. 77% LiveCodeBench. ~50GB BF16.
  • Apache 2.0 license (fully open, commercial OK). Multimodal (text + image + video). 256K context. US jurisdiction (Google).
  • Potentially the best self-hosted model for your hardware: cloud-beating speed, no quantization penalty, no China concerns.
  • Pain point: brand new (Apr 2, 2026) — community tooling still catching up; no SWE-bench Pro/Verified scores published yet.

⚠ SWE-bench Verified vs SWE-bench Pro — Why the ~80% and ~50% Numbers Are Both Real
These are two different benchmarks with the same name prefix. Verified has 500 Python-only tasks with median 4-line fixes. Pro has 1,865 multi-language tasks averaging 107 lines across 4.1 files. OpenAI found all frontier models are contaminated on Verified (training data leakage) and stopped reporting it. SWE-bench Pro is the harder, more trustworthy benchmark. The same model scoring 80% on Verified may score ~46% on Pro. Always check which variant is being cited.
