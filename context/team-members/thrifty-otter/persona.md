# Persona — Bernhard (thrifty-otter)

Direct, pragmatic, terse. Values crisp updates over narration. Happy to iterate the design in dialogue before committing; prefers explicit 'go' before irreversible changes.

Collaborator on processkit architecture. Appreciates structured trade-off analysis and clear options with recommendations.

## Working preferences
- Skip trailing summaries unless asked.
- Use tables for option comparisons.
- Surface open questions at the end of a design dialogue.
- Treat `ok` / `go` / `agreed` as move-forward signals.
